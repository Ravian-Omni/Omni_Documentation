# Web Automation in Chat

Ravian AI performs **one-off web tasks** like form filling, data extraction, or quick scraping — directly from your chat.

Ask it to "grab the latest pricing from X site" and get structured data back immediately.

---

## What this capability does

Chat Mode web automation handles:

- Navigate sites, fill forms, and extract specific data.
- Quick scraping of tables, lists, or detail pages.
- One-time logins and data pulls from dashboards.
- Simple multi-step web flows (find → extract → format).
- Return results as text, CSV, or screenshots in the chat.

!!! note
    Complex or repetitive web tasks should use Task Mode for planning and error handling.

---

## Typical workflows

Quick web tasks:

- Price monitoring or competitor data pulls.
- Form submissions or account updates.
- Lead list extraction from directories.
- Dashboard screenshots with key metrics highlighted.

---

## Example: Competitor pricing

1. Type in chat:

    ```text title="Prompt"
    Go to these 5 competitor sites and extract their pricing plans into a table.
    ```

2. Ravian visits each site, scrapes pricing data, and returns a clean comparison table.

3. Follow up: Email this table to my team — one click approval and it's sent.

4. Or save: Paste this into Google Sheets — done instantly.

Fast web automation without RPA setup or coding.

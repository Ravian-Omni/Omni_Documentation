# Web Automation & Browsing

Ravian AI acts as a **programmatic web operator**, handling complex browsing, form filling, and data extraction workflows.

Describe the web task; Ravian navigates, interacts, and delivers structured results.

---

## What this capability does

Task Mode web automation includes:

- Multi-site navigation with login, search, and filtering.
- Robust form filling across complex applications.
- Structured data extraction from tables, paginated results, and APIs.
- Error handling, retries, and validation of extracted data.
- Export results to CSV, databases, or other systems.

!!! warning
    Configure human approvals for form submissions or account actions.

---

## Typical workflows

Advanced web automation:

- Job application pipelines across multiple boards.
- Competitor price monitoring and alerting.
- Lead generation from directories and social platforms.
- E-commerce research and inventory tracking.

---

## Example: Job application pipeline

1. Create the task:

    ```text title="Prompt"
    Apply to 15 data engineering roles across LinkedIn, Naukri, and Indeed using my resume and preferences.
    ```

2. Ravian searches each platform, filters by criteria, fills forms, and queues applications for approval.

3. Approved applications are submitted; all activity logged in a tracking spreadsheet.

4. Daily runs maintain momentum without manual searching.

Scale web tasks without RPA infrastructure.
